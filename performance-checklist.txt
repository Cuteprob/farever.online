### 🔧 API 缓存头配置
**⚠️ 智能缓存策略 - 解决缓存问题**

**常见缓存问题及解决方案：**

1. **开发环境数据更新问题**
   - 问题：本地调试时更新数据库，页面总是加载缓存数据
   - 解决：开发环境禁用API缓存
   ```typescript
   // app/api/*/route.ts 中添加环境判断
   const cacheControl = process.env.NODE_ENV === 'development' 
     ? 'no-cache, no-store, must-revalidate'
     : 'public, max-age=300';
   ```

2. **数据加载不完整问题**
   - 问题：网速慢时加载不完整数据被缓存
   - 解决：只缓存成功的完整响应
   ```typescript
   // 只在数据完整时设置缓存
   if (data && data.length > 0) {
     headers: { 'Cache-Control': 'public, max-age=300' }
   } else {
     headers: { 'Cache-Control': 'no-cache' }
   }
   ```

3. **评分组件实时更新问题**
   - 问题：用户评分后组件显示旧数据
   - 解决：评分数据使用ETag + 条件缓存
   ```typescript
   // 评分API返回ETag，客户端检查更新
   const etag = `"${gameId}-${Date.now()}"`;
   headers: {
     'ETag': etag,
     'Cache-Control': 'private, max-age=0, must-revalidate'
   }
   ```

**分层缓存策略实施：**

□ **为不同API应用差异化缓存**
   ```typescript
   // 游戏列表 - 中等缓存（5分钟）
   // app/api/getGamesByCategory/route.ts
   headers: { 
     'Cache-Control': process.env.NODE_ENV === 'development'
       ? 'no-cache'
       : 'public, max-age=300, stale-while-revalidate=600' 
   }
   
   // 评分数据 - 短缓存（1分钟）
   // app/api/games/rating/route.ts  
   headers: {
     'Cache-Control': process.env.NODE_ENV === 'development'
       ? 'no-cache' 
       : 'private, max-age=60, must-revalidate',
     'ETag': `"${gameId}-${timestamp}"`
   }
   
   // 评论数据 - 长缓存（10分钟，因为需要审核）
   // app/api/comments/route.ts
   headers: {
     'Cache-Control': process.env.NODE_ENV === 'development'
       ? 'no-cache'
       : 'public, max-age=600, stale-while-revalidate=1200'
   }
   
   // 静态游戏信息 - 长缓存（1小时）
   // app/api/[game_id]/route.ts
   headers: {
     'Cache-Control': process.env.NODE_ENV === 'development'
       ? 'no-cache'
       : 'public, max-age=3600, stale-while-revalidate=7200'
   }
   ```

□ **客户端缓存清理机制**
   ```typescript
   // components/StarRating.tsx 中添加
   const clearRatingCache = async (gameId: string) => {
     // 清理浏览器缓存
     if ('caches' in window) {
       const cache = await caches.open('api-cache');
       await cache.delete(`/api/games/rating?gameId=${gameId}`);
     }
     
     // 发送缓存失效通知
     window.dispatchEvent(new CustomEvent('ratingCacheInvalidate', {
       detail: { gameId, timestamp: Date.now() }
     }));
   };
   ```

影响: 解决缓存问题，提升响应速度 70%，确保数据实时性

□ **实施差异化 API 缓存策略**

   ```typescript
   // 智能缓存策略配置
   const CACHE_STRATEGIES = {
     // 开发环境统一禁用缓存
     development: 'no-cache, no-store, must-revalidate',
     
     // 生产环境分层缓存
     production: {
       games_list: 'public, max-age=300, stale-while-revalidate=600',    // 5分钟
       ratings: 'private, max-age=60, must-revalidate',                   // 1分钟，必须验证

       static_data: 'public, max-age=3600, stale-while-revalidate=7200'   // 1小时
     }
   };
   
   // 使用示例
   const getCacheHeaders = (type: string) => {
     if (process.env.NODE_ENV === 'development') {
       return { 'Cache-Control': CACHE_STRATEGIES.development };
     }
     return { 'Cache-Control': CACHE_STRATEGIES.production[type] || 'no-cache' };
   };
   ```

   ```typescript
   // 检查If-None-Match头
   const ifNoneMatch = request.headers.get('if-none-match');

     return new NextResponse(null, { status: 304, headers: { 'ETag': etag } });
   }
   ```
